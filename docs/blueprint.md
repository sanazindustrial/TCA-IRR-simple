# **App Name**: Startup Compass

## Core Features:

- TCA Scorecard: Evaluate startups based on 12 categories (Product, Market, Team, etc.) with weighted scoring. It uses uploaded documents, pitch decks, and questionnaires as tool inputs. It also integrates external databases (Crunchbase, Pitchbook) to provide an evaluation and composite score.
- Risk Flags: Identify and flag risks across 14 domains (Market, Team, Financial, etc.) with red/yellow/green indicators and AI recommendations for mitigation, based on compliance checklists and uploaded docs. The AI reviews regulatory DBs to assist.
- Benchmark Comparison: Compare startup performance against sector averages, top performers, and competitors using percentile ranks and spider charts. The analysis involves sector averages and competitor datasets.
- Macro Trend Alignment: Assess startup alignment with external trends (Political, Economic, Social, Technological, Environmental, Legal) using a PESTEL dashboard and trend overlay scores based on data ingested from newsfeeds and trend databases (World Bank, IMF, OECD)
- Gap Analysis: Identify performance gaps compared to peers and benchmarks, highlighting priority areas and quick wins, with a recommended action plan that includes a roadmap, and utilizes uploaded decks.
- Growth Classifier: Classify startup growth potential using an ensemble of machine learning models (XGBoost, Random Forest, Neural Network, SVM, Gradient Boosting, Ensemble Voting), with scenario simulations (base/best/worst) and confidence intervals.
- Funder Fit Analysis: Match startup profiles with investor expectations, assessing funding readiness and providing valuation analysis, with an investor shortlist. It also evaluates potential investor thesis alignment, using VC and angel databases.

## Style Guidelines:

- Primary color: Strong purple (#9C27B0) for a sophisticated and innovative feel.
- Background color: Light purple (#F3E5F5) to ensure elements don't fight with one another for the user's attention.
- Accent color: Blue (#3F51B5) to complement purple, create depth, and focus attention.
- Font: 'Inter' (sans-serif) for a clean, modern, and neutral user interface.
- Employ a grid-based layout for structured and responsive design.
- Use consistent and clear icons to represent different evaluation categories.
